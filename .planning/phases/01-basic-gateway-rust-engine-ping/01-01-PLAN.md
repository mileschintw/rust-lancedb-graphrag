Output:
- Shared Protobuf contract and Buf generation configuration.
- Go HTTP API Gateway with Chi, Zap logging, pgx/v5 database connection pool scaffolding, and a health check handler.
- Rust RAG Engine gRPC server with Tokio and tracing.
- Docker Compose setup for Postgres and Jaeger.

<verification>
Verify the end-to-end flow:
1. Start containers: `docker compose up -d`
2. Start Rust RAG Engine: `cargo run --manifest-path engine/Cargo.toml`
3. Start Go Gateway: `go run gateway/main.go`
4. Send health check query:
   `curl -i http://localhost:8080/health`
   Verify response is HTTP 200 with JSON payload containing status ok and engine latency.
5. Terminate Rust engine process and send health check query again:
   `curl -i http://localhost:8080/health`
   Verify response is HTTP 503 Service Unavailable.
</verification>
